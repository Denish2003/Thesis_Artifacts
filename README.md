# Thesis Artifacts

This repository contains the source-code for the software artifacts produced for the thesis. At the time of writing, the codebase is organized into three main components.

- Data Export Extension: the browser extension used to capture structured ChatGPT conversations directly from the client interface.
- Data Donation Platform : the Streamlit application used for consent, review, redaction, cleaned export, and optional data donation.
- Data Analysis Pipeline: the notebooks and supporting scripts used to normalize WildChat data, construct the companionship corpus, and generate the reported analyses and figures.
